# 月度 Git 工作汇总 Few-shot 样例

## 用途

本文件用于给 AI 提供稳定的输出风格参考。样例只表达格式和语气，不代表每月都要复用相同项目、模块或统计数据。

### 输入摘要

```json

  "project": "BluettiAppH5",
  "branch": "develop",

      "files": 4,
      "insertions": 499,
      "deletions": 55,
      "topic": "附件与下载链路优化"
    },
    {
      "hash": "2b3c4d5",
      "time": "2026-04-22 15:42:43 +08:00",
      "subject": "feat: 调整详情页的布局，点击目录/文档类型后，展示首页的列表视图；",
```

## 样例二：事实版总结

### 输入摘要

```json
{
  "report_month": "2026-04",
  "totals": {
    "commits": 81,
    "projects": 5,
    "file_changes": 336,
    "unique_files": 209,
    "insertions": 8982,
    "deletions": 2421
  },
  "top_projects": [
    { "name": "go-hire-sale", "commits": 38 },
    { "name": "plm", "commits": 21 },
    { "name": "BluettiAppH5", "commits": 14 }
  ],
  "topics": [
    "订单审核与订单详情链路补齐",
    "客户审核与统计分析页面调整",
    "PLM 流程看板、待办、任务能力完善",
    "营销物料 H5 下载、布局、国际化和图标回显优化",
    "NetSuite 一物多码、发票语言及脚本数据适配"
  ]
}
```

### 输出样例

```markdown
2026 年 4 月共在 5 个项目完成 81 次分支提交，工作重心集中在 go-hire-sale、plm 与 BluettiAppH5。主要产出包括订单审核与订单详情链路补齐、客户审核与统计分析页面调整、PLM 流程看板/待办/任务能力完善，以及营销物料 H5 的下载、布局、国际化和图标回显优化；同时还处理了 NetSuite 一物多码、发票语言及脚本数据适配。
```

## 样例三：KPI 量化版总结

### 输入摘要

```json
{
  "report_month": "2026-04",
  "totals": {
    "commits": 81,
    "projects": 5,
    "file_changes": 336,
    "unique_files": 209,
    "insertions": 8982,
    "deletions": 2421
  },
  "top_projects": [
    { "name": "go-hire-sale", "commits": 38 },
    { "name": "plm", "commits": 21 },
    { "name": "BluettiAppH5", "commits": 14 }
  ],
  "business_lines": [
    "交易审批",
    "流程协同",
    "营销物料",
    "权限与国际化",
    "下载链路与环境发布"
  ]
}
```

### 输出样例

```markdown
按交付量化，本月覆盖 5 个项目、81 次提交，累计触达 336 次文件改动，覆盖 209 个唯一文件，代码变更 11403 行（新增 8982 / 删除 2421）。其中 go-hire-sale 38 次、plm 21 次、BluettiAppH5 14 次，核心投入集中在交易审批、流程协同与营销物料三条业务线，同时兼顾权限、国际化、下载链路与环境发布优化。
```

## 样例四：不应输出的内容

以下表达不应出现在最终月报中：

- `显著提升用户满意度`：Git 数据无法证明用户满意度。
- `成功上线并获得业务认可`：除非输入数据包含上线和反馈证据。
- `为公司创造明显收益`：Git 数据无法证明收入结果。
- `完成全部测试验证`：除非输入数据明确包含测试验证提交。
- `优化了所有相关页面`：范围过大，容易超出 Git 事实。

## 样例五：推荐中性表达

可优先使用以下表达保持客观一致：

- `主要产出包括...`
- `工作重心集中在...`
- `同时处理了...`
- `围绕...持续完善...`
- `覆盖...、...和...等内容`
- `按交付量化...`
- `累计触达...`
- `核心投入集中在...`
